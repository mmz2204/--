                📚 JSON教程
            

            
            

                
📖 什么是 JSON？

                

                    JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。它基于JavaScript语法，但独立于编程语言，易于人阅读和编写，同时也易于机器解析和生成。
                

            


            

                
📌 JSON 基本语法

                

                    数据类型：
                    
    
                        
  • string - 字符串，使用双引号
    • 
                        
  • number - 数字，整数或浮点数
    • 
                        
  • boolean - 布尔值，true 或 false
    • 
                        
  • null - 空值
    • 
                        
  • array - 数组，使用方括号 []
    • 
                        
  • object - 对象，使用花括号 {}
    • 
                    

                

            


            

                
💡 JSON 示例

                

                    一个简单的 JSON 对象示例：
                    
{
  "name": "张三",
  "age": 25,
  "isStudent": true,
  "hobbies": ["阅读", "编程", "运动"],
  "address": {
    "city": "北京",
    "street": "朝阳区"
  }
}

                

            


            

                
⚠️ 常见错误

                

                    
    
                        
  • 键名必须使用双引号，不能使用单引号
    • 
                        
  • 字符串必须使用双引号
    • 
                        
  • 数组或对象最后一个元素后面不能有逗号
    • 
                        
  • 不能有注释
    • 
                        
  • 不能有尾随逗号
    •
